# PSeInt - Metodologia de la Programacion

Ejercicios de pseudocódigo realizados en clase con [PSeInt](https://pseint.sourceforge.net/), agrupados por nivel de dificultad.

## Estructura

| Carpeta | Nivel | Archivos | Descripción |
|---------|-------|----------|-------------|
| `Faciles/` | Básico | 9 | Algoritmos introductorios: arrays fijos, bucles, condicionales, funciones básicas |
| `Medio/` | Intermedio | 1 | Arrays dinámicos, funciones con parámetros de tamaño, validación de entrada |
| `dificil/` | Avanzado | 1 | Programa modular completo: menú interactivo, subprocesos, múltiples funciones |

## Temas cubiertos

- Entrada/salida (`Escribir`, `Leer`)
- Estructuras de control: `Si`/`SiNo`, `Segun`, `Para`, `Repetir`
- Funciones (`Funcion`) y subprocesos (`SubProceso`)
- Arreglos (tamaño fijo y dinámico)
- Búsqueda lineal, números primos, par/impar, promedio, calculadora
- Validación de entrada y programación defensiva

## Total

11 archivos `.psc` organizados en una progresión didáctica: conceptos simples → generalización → programa integrado con menú.
